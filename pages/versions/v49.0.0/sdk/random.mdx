> **warning** **Deprecated:** This module will no longer be available in SDK 49. Use [nxenv-crypto](crypto.mdx) instead.

`nxenv-random` provides a native interface for creating strong random bytes. With `Random` you can create values equivalent to Node.js core `crypto.randomBytes` API. `nxenv-random` also works with `nxenv-standard-web-crypto`, which implements the W3C Crypto API for generating random bytes.
