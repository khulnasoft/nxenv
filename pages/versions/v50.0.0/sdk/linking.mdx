`nxenv-linking` provides utilities for your app to interact with other installed apps using deep links. It also provides helper methods for constructing and parsing deep links into your app. This module is an extension of the React Native [Linking](https://reactnative.dev/docs/linking.html) module.

For a more comprehensive explanation of how to use `nxenv-linking`, refer to the [Linking into other apps](/linking/into-other-apps/).
